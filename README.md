# Pretty Little Skills

A collection of agent skills — for wherever you want to put them. Some of these are for work, some are for personal life stuff — take a look and be open minded.

Each skill lives in `plugins/<skill-name>/skills/<skill-name>/SKILL.md`.
That layout also works as a Claude Code plugin marketplace, but the skills
themselves are plain markdown and portable anywhere skills are supported.

## Pretty Little Skills

- [in-english](plugins/in-english/skills/in-english/SKILL.md) — translates jargon-heavy tech/AI language into plain terms.
- [step-explainer](plugins/step-explainer/skills/step-explainer/SKILL.md) — builds an interactive HTML artifact that teaches a concept by simulating one concrete instance of it.
- [make-it-fun](plugins/make-it-fun/skills/make-it-fun/SKILL.md) — reframes a stressful upcoming task as a game instead of a life-or-death event, to defuse anxiety.
- [handover-webchat](plugins/handover-webchat/skills/handover-webchat/SKILL.md) — packages the current web chat conversation into a portable handover kit so a new chat can pick the work up cold.
- [get-started-webchat](plugins/get-started-webchat/skills/get-started-webchat/SKILL.md) — cold-starts a new web chat from a handover kit, reading it in the right order and getting straight back to work.
