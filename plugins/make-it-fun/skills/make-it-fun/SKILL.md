# Make It Fun

Anxiety feeds on importance. The fix isn't to lower the actual stakes of the task — it's to stop treating it like a matter of life and death. Do the interview *for fun*. Submit the documents *for fun*. The task stays real; only the weight comes off.

## How to build one

1. **Read the task, spot the weight.** What's making it feel heavy — an evaluator judging them, an irreversible deadline, an unknown outcome? Name it to yourself, don't repeat it back solemnly.
2. **Pick a game that fits its shape.** Match the genre to the task, don't force one template:
   - Being evaluated (interview, audition, review) → a performance, a boss fight, a bit.
   - Paperwork/bureaucracy (visa docs, forms, applications) → a scavenger hunt, a puzzle, collecting items.
   - A first/unfamiliar situation (new job, new place) → an expedition, exploring a new map.
   - A hard conversation → a side quest, a negotiation minigame.
   If none fit, invent one rather than stretching a bad match.
3. **Write it short and playful.** A few punchy lines, not an essay: a win condition, a score, a made-up rule or two, maybe a title. Wit over whimsy — it should make them smirk, not read like a children's book.
4. **Keep the real action legible underneath.** The game wrapper sits on top of the actual steps, not instead of them — someone should be able to tell what they're really about to go do.
5. **Close with the one real next step**, stated plainly, no game language, so the actual to-do doesn't get lost in the bit.
6. **Never act on their behalf.** Only write the reframe. Don't send, submit, email, or schedule anything — hand it back for them to read first.
