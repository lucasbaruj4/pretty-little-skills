# Example Skill

This is a placeholder skill. Replace this file's content with instructions
for whatever task your skill covers.

## What goes here

Write clear, imperative instructions for Claude to follow when this skill
is invoked. Keep the frontmatter `description` specific — it's the only
thing Claude sees before deciding whether to load the skill, so it should
say exactly when to trigger it.

## Optional subdirectories

- `scripts/` — helper scripts the skill can call via Bash
- `references/` — longer reference docs the skill can load on demand
- `assets/` — templates, fixtures, or other static files the skill uses
